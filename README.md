# 🎨 Ian Ryu — Canvas Drawing App

A lightweight browser-based drawing and whiteboard application built using **HTML5 Canvas and Vanilla JavaScript**.  
It provides a simple but powerful workspace for sketching, diagramming, and visual thinking — similar to tools like Excalidraw or Figma whiteboards.

---

## ✨ Features

### 🧩 Drawing Tools
- Rectangle tool
- Ellipse tool
- Line tool
- Arrow tool
- Pen (freehand drawing)
- Text tool
- Eraser
- Selection tool
- Pan tool

### 🎨 Styling Controls
- Stroke color palette
- Custom color picker
- Brush sizes (S, M, L, XL)
- Opacity slider

### 🔄 Canvas Features
- Infinite zoomable workspace
- Pan and drag navigation
- Undo / Redo system
- Smooth tool switching

### 📤 Export
- Export canvas as PNG image

### ⌨️ Keyboard Shortcuts
- V — Select tool  
- H — Pan tool  
- R — Rectangle  
- E — Ellipse  
- L — Line  
- A — Arrow  
- P — Pen  
- T — Text  
- X — Eraser  
- Ctrl + Z — Undo  
- Ctrl + Y — Redo  

---

## 🧠 How It Works

This app is built using:
- HTML5 `<canvas>` for rendering
- Vanilla JavaScript for logic and tool system
- Event-based mouse interactions for drawing
- Internal history stack for undo/redo
- Canvas transform system for zoom and pan
- Text overlay input for typing on canvas
